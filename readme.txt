== Description ==

Allow your users or customers to connect with you via WhatsApp simply by clicking on a button.
 Users can contact you directly on your WhatsApp number and continue the conversation on WhatsApp Web,
 if the user is using a desktop, or open the WhatsApp application if user is on a mobile device.
For more information visit https://hackforme.com

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
1. after activation you should see a whatsapp page in admin panel. you can set further settings from there
